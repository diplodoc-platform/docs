# Функциональные возможности

## Расширенный Markdown

Поддержка [расширенного синтаксиса Markdown](syntax/index.md), включая:
- кодовые блоки;
- изображения, видео и диаграммы;
- таблицы и списки.

## Комбинирование ручной и генерируемой документации

Поддержка генерации из [OpenAPI-спецификаций](guides/openapi.md), а также автогенерация из Swift и Objective-C.

## Поиск по документации

Поиск по документации на базе [Lunr.js](project/lunr.md), поддержка интеграции с облачным поиском [Algolia](project/algolia.md). Собственный поиск на базе opensearch для серверной версии.

## Локализация

Нативная поддержка [извлечения текста](tools/docs/translate.md) для различных систем перевода, таких как SmartCAT, Weblate, YaCAT и Яндекс Переводчик.

## Конструктор страниц

Нативная поддержка [Page Constructor](https://github.com/gravity-ui/page-constructor) для создания лендингов с каруселями, фонами, плавно анимированными карточками и другими типами блоков.

## Режим Right-to-Left

Режим Right-to-Left (RTL) для языков с письмом справа налево (например: арабский, иврит).

## Визуализация данных

Поддержка инструментов визуализации:
- [Mermaid](custom-plugins/mermaid.md).
- LaTeX.
