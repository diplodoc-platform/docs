# Yandex Flavored Markdown

**Yandex Flavored Markdown (YFM)** — диалект Markdown с дополнительным набором инструментов для трансформации Markdown в HTML и сборки документационных проектов.

Особенности:

* Совместимость со стандартом [CommonMark Spec](https://spec.commonmark.org/).
* Возможность [подключения](./plugins/external.md) сторонних плагинов markdown-it.
* Безопасность: HTML экранируется по умолчанию.
* Динамическая валидация.
* Сборка документационного проекта.

Синтаксис YFM основан на [CommonMark Spec](https://spec.commonmark.org/) и дополнен элементами из других языков разметки и шаблонизаторов.
