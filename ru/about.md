# О Diplodoc

Diplodoc — платформа для работы с документацией на основе подхода «docs as code».

## Варианты использования

{% list tabs %}

- Серверная версия

  Вариант с хостингом на стороне сервиса:
  - Обработка и рендеринг контента на ресурсах Diplodoc.
  - Интеграция с GitHub для работы с Markdown-файлами: автоматическая генерация документации на домене [diplodoc.com](https://diplodoc.com/).
  - Поддержка проксирования с пользовательских доменов.

- Статическая документация

  Вариант с локальной сборкой:
  - Генерация статических HTML-страниц через CLI.
  - Размещение на любых хостинговых платформах.
  - Интеграция с CI/CD пайплайнами.

{% endlist %}
