# О Diplodoc

Diplodoc — платформа для работы с документацией на основе подхода «docs as code».

## Варианты использования

| Серверная версия | Статическая документация |
|-----------------|-------------------------|
| **Вариант с хостингом на стороне сервиса:** | **Вариант использования через интерфейс командной строки (CLI):** |
| - Обработка и рендеринг контента на ресурсах Diplodoc. | - Генерация статических HTML-страниц через CLI. |
| - Интеграция с GitHub для работы с Markdown-файлами: автоматическая генерация документации на домене [diplodoc.com](https://diplodoc.com/). | - Размещение на любых хостинговых платформах. |
| - Поддержка проксирования с пользовательских доменов. | - Интеграция с CI/CD пайплайнами. |
