# Отправка целей в Метрику

После [подключения Метрики](../../settings.md#analytics-metrika) в проект в счётчик начинают отправляться [цели](https://yandex.ru/support/metrica/ru/general/goals) на события в интерфейсе документации.

{% note info %}

Чтобы цели сохранялись в Метрике, необходимо [добавить их в счётчик по инструкции](https://yandex.ru/support/metrica/ru/general/goal-js-event).

{% endnote %}

## Поддерживаемые цели

`DOCS_EDIT-IN-VCS_CLICK`
:   Нажатие на кнопку-ссылку "Редактировать".

`DOCS_FOOTER-NEXT_CLICK`
:   Переход на следующую страницу по ссылке "Следующая" в конце статьи.

`DOCS_FOOTER-PREV_CLICK`
:   Переход на предыдущую страницу по ссылке "Предыдущая" в конце статьи.

`DOCS_HEADER-LOGO_CLICK`
:   Нажатие на логотип.

`DOCS_HEADER-SANDWICH-MENU_CLICK`
:   Раскрытие меню оглавления в мобильной версии.

`DOCS_HEADER-SETTINGS_CLICK`
:   Открытие пункта меню "Настройки".

`DOCS_LANG_CLICK`
:   Открытие меню выбора языка.

`DOCS_WIDESCREEN-MODE_CLICK`
:   Включение/выключение полноэкранного режима.

`DOCS_HIDE-PAGE-NAV_CLICK`
:   Включение/выключение навигации по текущей статье.

`DOCS_FONTSIZE-MODE_CLICK`
:   Изменение размера текста.

`DOCS_READING-MODE_CLICK`
:   Переход в режим чтения.

`DOCS_SEARCH_CLICK`
:   Нажатие на поле ввода поискового запроса.
