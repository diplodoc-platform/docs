# Генерация llms.txt

##llms.txt## — это специальный текстовый файл в формате Markdown, который размещается в корне сайта и используется ИИ-системами для улучшенной навигации по страницам. Его главная задача — помочь правильно понять структуру и смысл сайта, выделить ключевую информацию и корректно её интерпретировать при генерации ответов.

##llms-full.txt## — это специальный текстовый файл в формате Markdown, который размещается в корне сайта и содержит полное текстовое содержимое всех страниц сайта для быстрого получения контента ИИ-системами.

В Diplodoc поддержана генерация файлов ##llms.txt## и ##llms-full.txt## для документации: необходимо добавить в `.yfm` параметр [`llms: enabled: true`](../settings.md#llms) или выполнить сборку с параметром `--llms`.

Структура собранного файла ##llms.txt##:

```markdown
# Название проекта

> Содержимое поля `llms: description` из файла .yfm

## Documentation

- [Заголовок статьи 1](ссылка-на-статью-1)
- [Заголовок статьи 2](ссылка-на-статью-2): содержимое поля description из метаданных статьи 2
- [Заголовок статьи 3](ссылка-на-статью-3): содержимое поля description из метаданных статьи 3
...

---

For more comprehensive documentation, see [llms-full.txt](llms-full.txt)
```

Особенности:

- для каждого ##toc.yaml## генерируется отдельная пара файлов ##llms.txt## / ##llms-full.txt##;
- поле `llms: description` из `.yfm` добавляется в качестве описания проекта;
- описание содержимого каждой статьи для ##llms.txt## берется из [поля ##description## в метаданных](../project/meta.md#description) страницы.
