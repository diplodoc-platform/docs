### Элемент по умолчанию

{% note info %}

Поддерживается в табах и радиобаттонах.

{% endnote %}

Если нужно, чтобы выбранный элемент был раскрыт по умолчанию, добавьте атрибут `{selected}` к нужному пункту.

{% list tabs %}

- Пример с табами

  ```markdown
  {% list tabs %}

  - Название пункта 1

    Текст

  - название пункта 2 {selected}

    Пункт будет открыт по умолчанию.

  {% endlist %}
  ```

  **Результат:**

  {% list tabs %}

  - Название пункта 1

    Текст

  - название пункта 2 {selected}

    Пункт будет открыт по умолчанию.

  {% endlist %}

- Пример с радиобаттонами

  ```markdown
  {% list tabs radio %}

  - Название пункта 1

    Текст.

  - Название пункта 2 {selected}

    Пункт будет открыт по умолчанию.

  {% endlist %}
  ```

  **Результат:**

  {% list tabs radio %}

  - Название пункта 1

    Текст.

  - Название пункта 2 {selected}

    Пункт будет открыт по умолчанию.

  {% endlist %}

{% endlist %}
