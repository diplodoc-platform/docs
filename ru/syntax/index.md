# Синтаксис

Diplodoc использует синтаксис **Yandex Flavored Markdown (YFM)** — диалект Markdown с дополнительным набором инструментов для трансформации Markdown в HTML и сборки документационных проектов.

Особенности:

* Совместимость со стандартом [CommonMark Spec](https://spec.commonmark.org/).
* Возможность [подключения](../plugins/external.md) сторонних плагинов markdown-it.
* Безопасность: HTML экранируется по умолчанию.
* Динамическая валидация.
* Сборка документационного проекта.

Для оформления используется:
* [базовая разметка](./base.md);
* [ссылки](./links.md);
* [списки](./lists.md);
* [простые](./tables/gfm.md) и [многострочные таблицы](./tables/multiline.md);
* [заметки](./notes.md);
* [каты, табы и радиобаттоны](./interactive-elements/index.md);
* [изображения и видео](./media.md);
* [фрагменты кода](./code.md);
* [всплывающие подсказки](./term.md);
* [переменные](./vars.md).

Кроме элементов, включенных в пакет, доступно подключение [дополнительных возможностей](./additional.md) через плагины markdown-it.
