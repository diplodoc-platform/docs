# Каты

{% note info %}

Содержимое катов может включать любую разметку YFM.

{% endnote %}

Используйте каты, чтобы скрыть часть контента. Например, дополнительную информацию или длинные блоки кода.

```markdown
{% cut "Заголовок ката" %}

Контент, который отобразится по нажатию.

{% endcut %}
```

**Результат:**

{% cut "Заголовок ката" %}

Контент, который отобразится по нажатию.

{% endcut %}
