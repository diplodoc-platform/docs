# Дропдаун

Дропдаун — выпадающий список. Используйте его для группировки элементов со скрытым контентом.

```markdown
{% list tabs dropdown %}

- Название пункта 1

  Контент для пункта 1.

- Название пункта 2

  Контент для пункта 2.

{% endlist %}
```

**Результат:**

{% list tabs dropdown %}

- Название пункта 1

  Контент для пункта 1.

- Название пункта 2

  Контент для пункта 2.

{% endlist %}
