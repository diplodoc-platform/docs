# Аккордеон

Аккордеон позволяет свернуть или раскрыть содержимое по вашему выбору, подходит для организации больших объемов информации.

{% include [blank-lines](../../_includes/blank-lines-note.md) %}

```markdown
{% list tabs accordion %}

- Название пункта 1

  Контент для пункта 1.

- Название пункта 2

  Контент для пункта 2.
 
- Название пункта 3

  Контент для пункта 3.

{% endlist %}
```

**Результат:**

{% list tabs accordion %}

- Название пункта 1

  Контент для пункта 2.
 
- Название пункта 3

  Контент для пункта 3.

{% endlist %}
