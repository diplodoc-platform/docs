# Элемент по умолчанию

{% note info %}

Задать элемент по умолчанию можно в следующих интерактивных элементах разметки: [табы](#tabs), [радиобаттоны](#radiobuttons), [дропдаун](#dropdawn) и [аккордеон](#accordion).

{% endnote %}

Если нужно, чтобы выбранный элемент был раскрыт по умолчанию, добавьте атрибут `{selected}` к нужному пункту.

## Табы {#tabs}

```markdown
{% list tabs %}

- Название таба 1

  Текст таба 1.

  * Можно использовать списки.
  * И **другую** разметку.

- Название таба 2 {selected}

  Текст таба 2.

{% endlist %}
```

**Результат:**

{% list tabs %}

- Название таба 1

  Текст таба 1.

  * Можно использовать списки.
  * И **другую** разметку.

- Название таба 2 {selected}

  Текст таба 2.

{% endlist %}

## Радиобаттоны {#radiobuttons}

```markdown
{% list tabs radio %}

- Название пункта 1

  Текст.

- Название пункта 2 {selected}

  Пункт будет открыт по умолчанию.

{% endlist %}
```

**Результат:**

{% list tabs radio %}

- Название пункта 1

  Текст.

- Название пункта 2 {selected}

  Пункт будет открыт по умолчанию.

{% endlist %}

##  Дропдаун {#dropdawn}

```markdown
{% list tabs dropdown %}

- Название пункта 1

  Контент для пункта 1.

- Название пункта 2 {selected}

  Контент для пункта 2.

{% endlist %}
```

**Результат:**

{% list tabs dropdown %}

- Название пункта 1

  Контент для пункта 1.

- Название пункта 2 {selected}

  Контент для пункта 2.

{% endlist %}

##  Аккордеон {#accordion}

```markdown
{% list tabs accordion %}

- Название пункта 1

  Контент для пункта 1.

- Название пункта 2 {selected}

  Контент для пункта 2.
 
- Название пункта 3

  Контент для пункта 3.

{% endlist %}
```

**Результат:**

{% list tabs accordion %}

- Название пункта 1

  Контент для пункта 1.

- Название пункта 2 {selected}

  Контент для пункта 2.
 
- Название пункта 3

  Контент для пункта 3.

{% endlist %}
