# Фрагменты кода1

Фрагмент кода можно добавить в текст или вынести в отдельный блок.

## В тексте {#inline}

Чтобы добавить фрагмент кода в текст, используйте символ `.

```markdown
`Фрагмент кода` в тексте.
```

**Результат:**

`Фрагмент кода` в тексте.

{% include [not_var](../_includes/not_var-info.md) %}

Префикс `not_var` работает **только** для фрагментов кода, состоящих из символов `.-|(),_`, `a-z`, `A-Z`, `0-9` и пробелов.

{% note tip "Совет" %}

Рекомендуется использовать не более 100 символов, так как текст в таком фрагменте не переносится. Для большего числа символов оформите код отдельным блоком.

{% endnote %}

## Отдельным блоком {#block}

Чтобы оформить фрагмент кода как отдельный блок, отделите его от остального текста с двух сторон символами ```.

Для подсветки синтаксиса укажите в начальной строке язык, на котором написан код. Например:

````markdown
```sql
  price= '2000'
  size= '24'  
  color= 'primary'
  variant= 'detailed' 
```
````

### Отображение номеров строк {#line-numbers}

Если необходимо включить отображение номеров строк в блоке кода, используйте ключевое слово `showLineNumbers`.

Пример использования:

````markdown
```sql showLineNumbers
  price = '2000'
  size = '24'  
  color = 'primary'
  variant = 'detailed' 
```
````

**Результат:**

```sql showLineNumbers
  price = '2000'
  size = '24'
  color = 'primary'
  variant = 'detailed'
```

### Перенос строк по умолчанию {#softwrap}

Чтобы включить перенос строк (soft wrap) по умолчанию в блоке кода, используйте ключевое слово `wrap`.

Пример использования:

````markdown
``` wrap
Очень длинная строка в блоке кода, которая точно не поместится в длину, если её искусственно не свернуть
```
````

**Результат:**

``` wrap
Очень длинная строка в блоке кода, которая точно не поместится в длину, если её искусственно не свернуть
```

### Префикс командной строки {#prompt}

Используйте параметр `prompt="<значение>"`, чтобы исключить префикс командной строки (`$`, `#`, `>>>`, `mysql>` и др.) из выделения и копирования через виджет.

**Пример использования:**

````markdown
```bash prompt="$"
$ npm install
$ npm run build
```
````

**Результат:**

```bash
$ npm install
$ npm run build
```

{% note tip "Совет" %}

Эта возможность особенно полезна для **сниппетов**. Если блок кода содержит только команды, без их вывода, то пользователь может скопировать всё содержимое одной кнопкой и вставить в терминал — не выделяя строки по отдельности.

{% endnote %}
