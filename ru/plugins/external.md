# Внешние плагины

## Подключение {#require}

Отличие внешних плагинов от встроенных в том, что перед подключением их нужно предварительно установить (`npm install`), а затем — добавить в секцию плагинов `mdit-plugins`.

Для внешних плагинов в поле `plugins` указывается имя плагина.

### Пример подключения {#example}

**Установка и настройка плагина [markdown-it-katex](https://www.npmjs.com/package/markdown-it-katex)**.

Плагин позволяет отображать математические формулы.

1. Установите пакет с плагином:

   ```bash
   npm install markdown-it-katex
   ```

2. Добавьте в файл `.yfm` следующую конфигурацию:

   ```yaml
   extensions:
   - name: mdit-plugins
      plugins:
         - "markdown-it-katex"
   ```

**Использование:**

```markdown
$\sqrt{3x-1}+(1+x)^2$
```

**Результат:**

$\sqrt{3x-1}+(1+x)^2$
