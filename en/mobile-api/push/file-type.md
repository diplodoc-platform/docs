# File types in push messages

Types of files that can be attached in a push message:

- image
- jpg
- jpeg2000
- tiff
- pict
- gif
- png
- qtif
- icns
- bmp
- ico
- raw
- svg
- livephoto
- audiovisual
- movie
- video
- audio
- mov
- mpeg
- mpeg2
- mpeg2stream
- mp3
- mp4
- mp4a
- m4p
- m4v
- avi
- aiff
- wav
- midi
