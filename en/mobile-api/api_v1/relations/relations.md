# Operators

| Operator | Description |
| ----- | ----- |
| == | Equal to |
| != | Not equal to |
| >= | Greater than or equal to |
| <= | Less than or equal to |
| report | Greater than |
| < | Less than |
| =@ | Is a substring |
| !@ | Not a substring |
| =~ | Regular expression |
| !~ | Not a regular expression |
| =* | Equal to, with wildcard search |
| !* | Not equal to, with wildcard search |
| =. | Found among the values |
| !. | Not found among the values |
